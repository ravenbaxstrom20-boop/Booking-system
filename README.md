# Bilingual Booking System

A bilingual service booking and lead conversion app built to capture, qualify, and organize customer requests in English and Spanish.

## Features
- English/Spanish language support
- Service selection and booking flow
- Lead capture form
- Booking summary panel
- Interaction history storage
- Responsive card-based UI

## Tech Stack
- Python
- Streamlit
- SQLite

## Purpose
Built as a practical booking and lead management system for service-based businesses, with a focus on usability, structured data capture, and clean UI design.

## Run Locally
```bash
streamlit run bilingual_booking_site.py
